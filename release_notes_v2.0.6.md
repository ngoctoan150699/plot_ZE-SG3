# ZE-SG3 Torque Acquisition v2.0.6

## Thay đổi chính

- Bổ sung luồng nhập Góc đo trên bản vẽ cho Inner Tie Rod + Oscillating Torque.
- Tự tính góc servo ±0.8X và ghi xuống PLC chỉ qua D102/D103.
- Tự điền Test Item Angle Range cho Inner Tie Rod / Oscillating Torque bằng ±0.64X.
- Tách profile servo Oscillating Torque khỏi Operating Torque.
- Tự xóa góc tạm sau khi chạy/lưu kết quả theo yêu cầu.

Asset đính kèm là file setup tạo bằng Inno Setup cho Windows.
